# MC-Technical-Tools

Herramienta técnica diseñada para el análisis y optimización de granjas de experiencia. El programa calcula el rendimiento operativo necesario para alcanzar el nivel 30, basándose en las mecánicas internas de spawn (versión Java).

Análisis de Granjas Automáticas: Estima el tiempo de espera (AFK) requerido para la acumulación de mobs.
Cálculo de Granjas Manuales: Determina la cantidad exacta de mobs a eliminar para cumplir el objetivo de XP.
Evaluación de Eficiencia: Clasifica el rendimiento de la estructura como "buena" o "lenta" según estándares técnicos.


El calculo se basa en la tasa de 1395 puntos de experiencia (progreso de nivel 0 a 30).
